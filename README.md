# Guru Diary Teaching Management System

## 📌 Project Overview
The Guru Diary Teaching Management System is a web-based application developed to simplify academic management within educational institutions. 

It allows administrators to efficiently generate and manage class timetables, while lecturers can record and track their daily teaching progress. The system improves coordination between faculty and administration and ensures smooth academic planning.

---

## 🚀 Key Features
- 🗓️ Admin-based timetable generation
- 👩‍🏫 Lecturer daily progress tracking
- 📊 Monitoring of syllabus completion
- 🔄 Improved coordination between admin and faculty
- 🧾 Organized academic records

---

## 🛠️ Technologies Used
- Python (Django Framework)
- HTML, CSS, JavaScript
- PostgreSQL Database
